# Website Traffic Analysis

## Objective
Analyze website traffic logs to understand user journeys, top landing pages, bounce rate, and referral sources.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Key Metrics
- Total Sessions
- Total Users
- Bounce Rate
- Average Session Duration

## Visualizations
- Top Landing Pages
- Top Exit Pages
- Referral Sources
- Top Countries
- Daily Traffic Trend
- Bounce Rate Analysis

## Key Insights
- Identified the most visited landing pages.
- Analyzed referral sources.
- Measured bounce rate and session duration.
- Studied user traffic by country.

## Recommendations
1. Improve high-bounce landing pages.
2. Optimize top exit pages.
3. Focus on high-performing referral sources.
4. Improve mobile user experience.
5. Strengthen calls-to-action to increase conversions.
